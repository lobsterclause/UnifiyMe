# Contributing to UnifiyMe

Thank you for your interest in contributing to UnifiyMe! This project aims to provide a high-integrity MCP server for UniFi controllers, and we welcome contributions that improve its stability, security, or features.

## How to Contribute

### Reporting Issues

- Use the GitHub Issue Tracker to report bugs or suggest features.
- Provide as much detail as possible, including logs, environment details, and reproduction steps.

### Pull Requests

1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Ensure your code follows the existing style (run `npm run lint`).
4. Ensure all tests pass (run `npm test`).
5. Add tests for new features where possible.
6. Submit a Pull Request with a clear description of the changes.

## Development Setup

1. Install dependencies: `npm install`
2. Run tests: `npm test`
3. Check linting: `npm run lint`
4. Build: `npm run build`

## Code of Conduct

Please be respectful and professional in all interactions within this project.
